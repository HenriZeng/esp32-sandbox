# ESP32 Learning Journey

This repository tracks my progress in embedded systems development using the **ESP32-E (N4)** module.

## Project Structure
- **01_hello_world**: Initial setup and serial communication testing.
- **02_wifi_features**: Exploring Wi-Fi station and access point modes.
- **03_ble_lab**: Experiments with Bluetooth Low Energy.

## Environment
- **Hardware**: ESP32-WROOM-32E (4MB Flash)
- **Framework**: Arduino / ESP-IDF (via PlatformIO)
- **IDE**: Cursor (AI-powered development)
